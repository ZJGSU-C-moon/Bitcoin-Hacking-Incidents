Mt.Gox ( 2011.03 ) —— 3000BTC

Gox 的热钱包被偷走了 8 万比特比特币，因为黑客复制了钱包文件。2011 年 5 月，黑客窃取了 30 万比特币，这些比特币被暂时存储在一个离线钱包中，该钱包位于一个不安全、可公开访问的网络驱动器上。然而，没过多久，黑客就紧张了起来，拿着 1%（3000 个比特币）的“保管费”后就退还了赃款。

Mt.Gox ( 2011.06 ) —— 2000 BTC

一名黑客侵入了 JedMcCaleb 的管理员账户，并且操纵了市场价格，从而导致市场出现了短暂的崩溃。事件结束后，黑客又成功窃取了 2000 个比特币。

MyBitcoin ( 2011.07 ) —— 78739 BTC

有关 MyBitcoin 被盗事件的信息几乎没有公布，许多人辩称，运营商 Tom Williams 将其作为一场骗局继续运营。这起盗窃案导致了 MyBitcoin 的关门。但在数字货币市场早期，MyBitcoin 曾是一家成功的比特币公司。

Bitomat.pl ( 2011.07 ) —— 17000 BTC

在服务器重启期间，远程亚马逊云服务上的 Bitomat.pl 的钱包被盗。钱包没有备份，但 Mt.Gox 后来给 Bitomat.pl 进行了兜底。最终，交易所的客户和资产所有者都没有因此事件蒙受任何损失。

Mt.Gox ( 2011.09 ) —— 77500 BTC

2011 年 9 月，一名黑客成功读取了 Mt.Gox 的数据库。黑客在该交易所开设了新账户，夸大了用户余额，并成功提取了 77500 个比特币，之后他们删除了大部分包含此类交易证据的日志。

Bitcoinica ( 2012.3 ) —— 61457 BTC

网络主机供应商 Linode 的服务器遭到黑客攻击，从而获得了访问存储在先驱交易所 Bitcoinica 上的比特币权限。这些事导致了最终的损失。

BitFloor ( 2012.09 ) —— 24000 BTC

当黑客能够访问该交易所钱包的未加密备份并将币转出去时，BitFloor 遭到了攻击。

丝绸之路 2.0 ( 2014.02 ) —— 44000 BTC

地下市场丝绸之路 2.0 的管理员 Defcon 注意到，在 2014 年 2 月，托管业务的资金从一个热钱包中被盗。“交易延展性”是比特币协议的一个问题，当时也影响了其他一些业务进展。虽说这是潜在的被盗原因之一，但也有很多人怀疑是内部工作人员所致。

Poloniex ( 2014.03 ) —— 97 BTC

2014 年 3 月，Poloniex 宣布，由于代码中存在一个此前未知的漏洞，该公司已成为黑客攻击的受害者。最终，该交易所告诉所有用户，他们的账户余额将减少 12.3%。

Cryptsy ( 2014.07 ) —— 13000 BTC

2016 年初，Cryptsy 在客户钱包中的 13000 枚比特币 (和 30000 枚 LTC) 被盗后崩溃。

Bitstamp ( 2015.01 ) —— 19000 BTC

黑客进入了 Bitstamp 的热钱包。被盗后，Bitstamp 开始将 98% 剩余比特币转入冷钱包。

LocalBitcoins ( 2015.01 ) —— 17 BTC

1 月 27 日有黑客在论坛上发帖，陈述如何通过其在线咨询 LiveChat 账户入侵，估计客户钱包有 17 个比特币丢失。

Evolution Darknet Marketplace ( 2015.03 ) —— 44000 BTC

2015 年 3 月，管理员“Kimble”和“Verto”出人意料地关闭了该交易所，这是一个在“丝绸之路 2.0”被占领后出现的暗网市场，因此，所有相关用户资金也在互联网上消失了。

Bitfinex ( 2016.08 ) —— 119756 BTC

2016 年 8 月 4 日，香港的比特币交易所 Bitfinex 遭黑客入侵，多达 119756 BTC 被盗，总价值约为 7500 万美元，比特币的价值因此大幅下跌 15%。

Yapizon ( 2017.04 ) —— 3381 BTC

2017 年 4 月 22 日，经韩国知名加密货币交易所 Yapizon 证实，黑客已经能够成功入侵其系统，盗取了四个独立的数字钱包，被盗比特币约为 3831 个，价值约 55 亿韩元。

Youbit ( 2017.04 ) —— 4000 BTC

2017 年 4 月，朝鲜黑客从从 Youbit 窃取了 4000 个比特币，按当时市值折算价值约 5300 万美元。

Bithumb ( 2017.06 ) —— 1900 BTC

黑客利用其已经掌握的 Bithumb 热钱包地址私钥，将 Bithumb 热钱包中的比特币不断转移至自己不同的钱包地址。

Nicehash ( 2017.12 ) —— 4000 BTC

2017 年 12 月，挖矿服务器Nicehash被黑客入侵，超过 4000 个比特币被盗，当时价值 6300 万美元。

Coinrail 和 Bithumb ( 2018.06 ) —— 5300 BTC

2018年6月，两家韩国交易所（Coinrail 和 Bithumb）的热钱包遭遇了攻击，其中 Coinrail 损失了 5300 个比特币（价值接近 4000 万美元），Bithumb 损失了近 3100 万美元。

Zaif ( 2018.09 ) —— 6000 BTC

2018 年 9 月 18 日，日本交易所 Zaif 宣布遭受黑客攻击，黑客从其热钱包中盗走了比特币、比特币现金和 MonaCoin 等巨额资金。

QuadrigaCX ( 2019.01 ) —— 51318 BTC

QuadrigaCX 的联合创始人于 2018 年 12 月 9 日去世，据称他是唯一有权使用该交易所私钥的人。不断变化的法律程序揭示了该交易所的资金管理不善和潜在的诈骗问题。这也导致人们呼吁加强对外汇业务的监管。

Binance ( 2019.05 ) —— 7000 BTC

2019 年 5 月 8 日，币安 BTC 热钱包发生被盗事件，据币安官方公告显示，这是一次大规模的系统性攻击，黑客获得了大量用户 API 密钥，谷歌验证 2FA 码以及其他相关信息。黑客在区块高度 575013 处从币安热钱包中盗取 7000 枚比特币。
